\noindent \textbf{External API contrast.} To test whether the main pattern is an artifact of local checkpoints, we ran five external API baselines on the same 560-example canonical split with the same prompt protocol (\texttt{decision\_first}, max\_tokens=64). \autoref{tab:day1-api-baseline}, \autoref{tab:day1-api-baseline-ci}, and \autoref{fig:api-baseline-comparison} show a nontrivial frontier spread: \texttt{gpt-5-chat-latest} reaches the best action accuracy (0.9286), \texttt{qwen-plus-latest} reaches the best utility (0.1571), and \texttt{qwen-turbo} remains strong at lower cost. We read these API rows as uncertainty-qualified point estimates rather than as a fine-grained significance ranking. The main claim is the persistent boundary difficulty: \texttt{gpt-5-chat-latest} still over-answers 29.17\% of false-premise cases even when overall action accuracy is high.

\noindent \textbf{Slice-balance sensitivity.} The canonical split keeps fixed action totals but leaves slice gaps (\texttt{answerable\_control} and \texttt{conflicting\_evidence}). To test whether this imbalance drives the API conclusion, we resumed only the 40 additional rows in a slice-balanced 600-example split for a frontier API, two stronger APIs, and one lower-cost API. Table~\ref{tab:day1-full-split-sensitivity} reports full-minus-canonical deltas under matched decoding, with bootstrap intervals for action accuracy and utility. For \texttt{qwen-plus-latest}, both \texttt{decision\_first} and \texttt{decision\_first\_guarded} have positive action-accuracy point deltas (+0.0096 and +0.0079), but only \texttt{decision\_first} improves utility (+0.0212), while guarded lowers utility (-0.0372). \texttt{gpt-5-chat-latest}, \texttt{gpt-4.1-mini}, and \texttt{qwen-turbo} show the same action-accuracy point-estimate direction (+0.0047, +0.0076, and +0.0107) while utility decreases (-0.0363, -0.0371, and -0.0357). Because the delta intervals overlap zero, we use the 600-example split as sensitivity evidence rather than as a replacement benchmark: the point estimates do not suggest a reversal, but the headline remains the canonical 560-example split.
